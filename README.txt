
# Protótipo de Previsão de Cancelamento de Alunos
Este protótipo utiliza um modelo de Machine Learning (Random Forest) para prever quais alunos têm maior risco de cancelar a matrícula na Academia Força Local.

## Como rodar
1. Instale o Python: https://www.python.org/downloads/
2. Instale as bibliotecas necessárias:
   ```
   pip install -r requirements.txt
   ```
3. Execute o aplicativo:
   ```
   streamlit run app.py
   ```
